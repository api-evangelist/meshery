# Meshery (meshery)
Meshery is the cloud native manager for Kubernetes and cloud native infrastructure. It is an extensible, self-service engineering platform that enables collaborative design, lifecycle and performance management of cloud native applications and infrastructure. Meshery exposes REST and GraphQL APIs for automating multi-cluster operations, design management, performance profiling, and integration with hundreds of CNCF and ecosystem projects.
